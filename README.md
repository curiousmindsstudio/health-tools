# Health Tools Hub 🏥

A simple, web-based collection of **digital health tools** including:

- **BMI Calculator** – Body Mass Index & target weight range
- **Age Calculator** – Find your age in years
- **Calorie Calculator** – Estimate daily calorie needs with recommendations

These tools are **fast, responsive, and easy to use**, perfect for personal use or as a portfolio project.

## 💡 Features

- Clean, **mobile-responsive design**  
- Accurate calculations based on standard formulas:  
  - BMI formula: weight / (height²)  
  - Mifflin-St Jeor for calories  
- **Interactive advice** based on results  
- Target weight range for BMI  
- Easy navigation between calculators

## ⚙️ Usage Instructions

1. Open any HTML file in your browser: `index.html`  
2. Navigate to the calculator you want to use  
3. Enter your details (weight, height, age, etc.)  
4. Click **Calculate**  
5. Read results and advice  

> Each tool shows interactive messages and target ranges where applicable.
